# AccountSystem - iOS Frontend

SwiftUI-based iOS application for the AccountSystem project.

## Requirements
- Xcode 13+
- iOS 15+
- Swift 5.5+

## Setup

1. Open `AccountSystem.xcodeproj` in Xcode
2. Select the AccountSystem target
3. Build and run on simulator or device

## Technologies
- SwiftUI
- Combine
- URLSession (API communication)
- CoreData (local storage)
