# AccountSystem - Web Frontend

React-based web application for the AccountSystem project.

## Setup

```bash
npm install
npm start
```

## Technologies
- React.js
- React Router
- Axios (API client)
- Material-UI or Tailwind CSS

## Project Structure

```
frontend-web/
├── public/
├── src/
│   ├── components/
│   ├── pages/
│   ├── services/
│   ├── hooks/
│   └── App.js
├── package.json
└── README.md
```
