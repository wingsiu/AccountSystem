# AccountSystem - WeChat Mini Program

WeChat mini program (小程序) version of the AccountSystem.

## Requirements
- WeChat Developer Tools
- Node.js 16+

## Setup

```bash
npm install
# Open in WeChat Developer Tools
```

## Technologies
- TypeScript/JavaScript
- WeChat Mini Program API
- Vant Weapp (UI components)

## Project Structure

```
frontend-miniprogram/
├── pages/
├── components/
├── services/
├── utils/
├── app.js
├── app.json
├── app.wxss
└── project.config.json
```

## Build & Deploy

1. Open in WeChat Developer Tools
2. Build project
3. Submit for review or deploy to staging
